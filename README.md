## Practive-next
### 如何执行
#### 1、将项目clone到本地，cd到Practive-next
```javascript
git clone https://github.com/kiik33/Practive-next.git
cd Practive-next
```
### 2、安装依赖
```javascript
npm install
或
yarn install
```
### 3、执行
```javascript
npm run dev
// npm run build（打包）
```
#### 4、打开浏览器浏览 http:/localhost:9000
